# 公開鍵暗号方式  
デジタル署名、SSHはこの仕組みを利用している。  
暗号を複合する側(=受信者側)が鍵のペアを作成する必要がある。  
受信者が鍵のペアを作成して公開鍵を送信者に渡して
暗号化して送ってもらう。
受信者は秘密鍵でそれを複合する。  

## 秘密鍵  
公開鍵暗号方式で使う、自分だけが持っている鍵。  

## 公開鍵  
みんなにばらまく鍵。南京錠側。  

## 実際に  
`$ ssh-keygen `で作成。  
* 秘密鍵`id_rsa`
* 公開鍵`id_rsa.pub`

# デジタル署名  
署名する側が自分の作った秘密鍵でデータを暗号化して
署名をしてもらう側は作ってもらった公開鍵で検証する。
